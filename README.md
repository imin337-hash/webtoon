# 🍌 Nano Banana Pro: Instatoon Generator

**인스타툰 '나노바나나프로' 제작을 위한 AI 프롬프트 생성기입니다.**

## 📖 프로젝트 소개
웹툰 창작의 3대 요소(스토리, 연출, 그림) 이론을 적용하여, 사용자가 선택한 옵션에 따라 **일관성 있는 10컷(4:5 비율)**의 미드저니(Midjourney)용 프롬프트를 자동으로 만들어줍니다.

## ✨ 주요 기능
1. **스토리 (Story):** 성장, 공감, 정보 등 테마 선택
2. **연출 (Direction):** 카메라 앵글 및 만화적 효과(SFX) 제어
3. **그림 (Art):** 감성 톤부터 팝(Pop)한 톤까지 색감 조절

## 🚀 사용 방법
1. 이 코드를 다운로드합니다.
2. 터미널에서 `pip install -r requirements.txt`를 실행합니다.
3. `streamlit run app.py`를 입력하면 웹페이지가 열립니다.
